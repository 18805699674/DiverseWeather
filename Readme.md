[402	超过访问次数或余额不足以支持继续访问服务，你可以充值、升级访问量或等待访问量重置。]

[404	查询的数据或地区不存在。]
[429	超过限定的QPM（每分钟访问次数），请参考QPM说明]
[500	无响应或超时，接口服务异常请联系我们]
